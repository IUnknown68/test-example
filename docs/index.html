test-example

Code example demonstrating code quality in the dimension of unit tests, documentation and similar topics. Besides that it doesn't provide any actual functionalities.

Installation

I will use yarn, but of course you can use the package manager of your choice.

git clone https://github.com/IUnknown68/test-example.git
cd test-example
yarn install

Commands

The following commands are available:

  • yarn lint: Lint the code. Base is airbnb.
  • yarn test: Run unit tests using jest.
  • yarn doc: Create documentation via jsdoc. Ouputs to ./docs.
  • yarn doc:md: Creates api.md using jsdoc-to-markdown.
